**主备份目录:** `/data/openclaw-memory-backup/`

## 🔧 如何使用

### 手动备份（推荐在重要对话后）

```bash
/root/.openclaw/workspace/scripts/backup-memory.sh
```

### 恢复记忆（重装系统后）

```bash
/root/.openclaw/workspace/scripts/restore-memory.sh
```

### 设置自动备份（可选）

编辑定时任务：
```bash
crontab -e
```

添加每天凌晨 2 点自动备份：
```
0 2 * * * /root/.openclaw/workspace/scripts/backup-memory.sh
```

### 每次对话后
- 小 Q 会自动更新当日日志

- 重要内容会同步到 `MEMORY.md`

- ❌ 不需要手动保存记忆

### 你可以做的（可选）

## 🚨 重装系统后恢复

### 场景 1：只重装系统，/data 分区保留

```bash
# 1. 重新安装 OpenClaw
# 2. 运行恢复脚本
/root/.openclaw/workspace/scripts/restore-memory.sh
# 3. 完成！记忆已恢复
```

### 场景 2：完整备份恢复

```bash
# 1. 从外部备份复制 /data/openclaw-memory-backup/ 到新机器
# 2. 重新安装 OpenClaw
# 3. 运行恢复脚本
/root/.openclaw/workspace/scripts/restore-memory.sh
```
