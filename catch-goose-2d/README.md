# 抓大鹅 2D - MVP

> 《抓大鹅》2D 简化版，用于市场验证的极简游戏 Demo

## 🎮 核心玩法

- **物品堆叠**：3 层随机堆叠
- **点击选取**：点击物品放入底部槽位
- **三消判定**：槽位中连续 3 个相同物品自动消除
- **60 秒倒计时**：时间内消除所有物品获胜

## 🚀 快速开始

### 方式一：直接打开（最简单）

```bash
# 直接用浏览器打开 index.html
open index.html  # Mac
start index.html # Windows
```

### 方式二：本地服务器

```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
http-server -p 8080

# 访问 http://localhost:8080
```
