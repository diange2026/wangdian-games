# 🎉 典哥的 Gitee 项目集合

## 📦 项目内容

### 1️⃣ 🐍 文件备份脚本 (`backup-files.py`)

**用途：** 帮你备份重要文件

**怎么用：**
1. 双击打开（需要安装 Python）
2. 输入要备份的文件夹路径
3. 输入备份到的位置
4. 回车，完成！

**适合场景：**
- 备份工作文档
- 备份照片
- 定期备份重要资料

---

### 2️⃣ 🌐 个人主页 (`index.html`)

**用途：** 展示个人信息的网页

**怎么用：**
1. 双击打开（用浏览器）
2. 可以看到个人信息
3. 可以分享给朋友看

**特点：**
- 漂亮的外观
- 手机电脑都能看
- 可以自定义内容

---

### 3️⃣ 📖 说明文档 (`README.md`)

**用途：** 介绍项目怎么用

**怎么看：**
- 在 Gitee 网页上直接看
- 或下载到本地用文本编辑器打开

---

### 4️⃣ 🎮 猜数字游戏 (`guess-number.html`)

**用途：** 休闲小游戏

**怎么玩：**
1. 用浏览器打开
2. 猜一个 1-100 的数字
3. 根据提示继续猜
4. 看几次能猜中！

**特点：**
- 简单好玩
- 可以反复玩
- 可以分享给朋友

---
