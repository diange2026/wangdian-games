## 📋 游戏规则

### 核心玩法
```
1. 系统随机生成一个 4 位数字（如 3847）
2. 每位数字不同（0-9）
3. 你有 10 次机会猜中
4. 每次猜测后系统提示：
   - xA = 数字和位置都正确
   - xB = 数字正确但位置错误
5. 猜中 4A0B = 胜利！
```

### 段位系统
| 段位 | 需要胜场 | 说明 |
|------|---------|------|
| 🥉 青铜 | 0 胜 | 新手入门 |
| 🥈 白银 | 5 胜 | 渐入佳境 |
| 🥇 黄金 | 15 胜 | 逻辑达人 |
| 💎 铂金 | 30 胜 | 推理高手 |
| 💠 钻石 | 50 胜 | 猜数大师 |
| 👑 王者 | 100 胜 | 传奇段位 |

### 每日挑战
- 每天刷新 3 道题目
- 完成挑战获得成就
- 连续完成有额外奖励
