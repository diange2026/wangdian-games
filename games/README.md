## 🎮 游戏规则

### 核心玩法
```
1. 系统随机生成一个 4 位数字（如 3847）
2. 每位数字不同（0-9）
3. 你有 10 次机会猜中
4. 每次猜测后系统提示：
   - xA = 数字和位置都正确
   - xB = 数字正确但位置错误
5. 猜中 4A0B = 胜利！
```

### 示例
```
答案：1234

你猜：1356 → 提示：1A1B
（1 位置对，3 数字对但位置错）

你猜：1278 → 提示：2A0B
（1 和 2 位置都对）

你猜：1234 → 提示：4A0B 🎉 胜利！
```
