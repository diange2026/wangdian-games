## 🚀 方案 A：企业微信（推荐）

### 优点
- ✅ 链接可点击
- ✅ 支持卡片消息
- ✅ 支持机器人
- ✅ 免费
- ✅ 5 分钟配置完成

### 配置步骤

#### 1️⃣ 创建企业微信群

```
1. 打开企业微信 APP
2. 点击「+」→ 发起群聊
3. 选择好友（至少 1 人）
4. 完成创建
```

#### 2️⃣ 添加群机器人

```
1. 进入群聊
2. 点击右上角「...」
3. 找到「群机器人」
4. 点击「添加」
5. 点击「新建」
6. 命名：猜数字大师助手
7. 头像：可选
8. 点击「添加」
```

#### 3️⃣ 复制 Webhook 地址

```
1. 添加成功后，会显示 Webhook 地址
2. 格式：https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx
3. 点击「复制」
4. 发送给小 Q
```

#### 4️⃣ 测试机器人

```
1. 在群里输入：/help
2. 机器人应该回复
3. 确认正常
```

## 📱 方案 B：微信公众号

### 优点
- ✅ 用户体验好
- ✅ 支持菜单栏
- ✅ 支持群发
- ✅ 可接广告

### 缺点
- ⚠️ 需要审核（1-3 天）
- ⚠️ 每天只能群发 1 次

### 注册流程

#### 1️⃣ 准备材料

```
- 身份证照片
- 手机号
- 邮箱
- 微信公众号名称
```

#### 2️⃣ 注册账号

```
1. 访问 https://mp.weixin.qq.com/
2. 点击右上角「立即注册」
3. 选择「订阅号」
4. 填写信息
5. 验证邮箱
6. 信息登记
7. 等待审核
```

#### 3️⃣ 配置菜单

```
1. 登录公众号后台
2. 左侧「内容与互动」→ 自定义菜单
3. 添加菜单：
   - 开始游戏
   - 游戏规则
   - 我的段位
4. 保存并发布
```
