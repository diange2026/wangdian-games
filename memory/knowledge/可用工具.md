## 📋 使用指南

### 文件操作
```bash
# 读取文件
read /path/to/file

# 写入文件
write /path/to/file "内容"

# 编辑文件（精确替换）
edit /path/to/file "旧文本" "新文本"

# 执行命令
exec "ls -la"
```

### 浏览器自动化
```bash
# 打开网页
agent-browser open https://example.com

# 获取页面快照
agent-browser snapshot -i --json

# 点击元素
agent-browser click @e2

# 填写表单
agent-browser fill @e3 "文本"
```

### 记忆操作
```bash
# 搜索记忆
memory_search "查询关键词"

# 读取记忆片段
memory_get /path/to/memory.md --from 10 --lines 20
```
