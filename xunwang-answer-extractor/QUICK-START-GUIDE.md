### 第一步：安装插件

#### 方法A：开发模式安装（推荐测试）
1. **下载插件**（如果你还没有）

   - 浏览器工具栏会出现插件图标

```bash

cd /root/.openclaw/workspace/xunwang-answer-extractor/
```

## 💡 核心功能使用

**使用场景**: 进入答案详情页面后

4. 下载文件

**使用方法**:

## 🐛 常见问题解决

### 问题1：插件不显示
**可能原因**:
1. 未开启开发者模式
2. 目录选择错误
3. manifest.json错误

**解决方案**:
```bash
# 检查manifest.json
cat manifest.json | python -m json.tool

# 重新加载插件
chrome://extensions/ → 点击刷新按钮
```

### 问题2：无法提取答案
**可能原因**:
1. 不在答案页面
2. 页面结构变化
3. JavaScript错误

**解决方案**:
```javascript
// 检查页面类型
console.log('当前页面:', window.location.href);

// 检查DOM结构
console.log('题目元素:', document.querySelectorAll('.question-item'));

// 调试模式
设置 → 高级设置 → 开启调试模式
```

### 问题3：导出文件问题
**可能原因**:
1. 没有提取数据
2. 格式不支持
3. 权限问题

**解决方案**:
```javascript
// 检查数据
console.log('提取的数据:', extractedData);

// 测试导出
console.log('导出格式支持:', exportFormats);

// 检查存储权限
chrome.permissions.contains({permissions: ['downloads']});
```
