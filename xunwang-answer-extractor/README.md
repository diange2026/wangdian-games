# 讯网题目答案提取器浏览器插件

## 📋 项目概述

专为讯网教学云平台（https://www.whxunw.com）设计的浏览器插件，帮助学员在补考时快速提取题目序号和标准答案，获得满分。

## 🎯 核心功能

### ✅ 已完成功能
1. **智能页面识别** - 自动识别讯网平台的答案详情页面
2. **一键提取答案** - 提取题目序号、题目内容、标准答案
3. **数据导出** - 支持Excel、CSV格式导出
4. **本地题库管理** - 保存提取的题目到本地数据库
5. **用户设置** - 可自定义提取参数和界面主题

### 🚀 规划功能
1. **自动登录** - 保存账号密码，自动登录平台
2. **批量处理** - 批量提取多门课程的题目
3. **智能分析** - 错题分析、知识点统计
4. **云同步** - 题库数据云端备份和同步
5. **商业变现** - 付费高级功能（团队协作、API接口等）

## 🛠️ 安装与使用

### 开发模式安装

1. **打开Chrome浏览器**
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的根目录
6. 插件安装完成

### 使用方法

1. **登录讯网平台**
   - 访问 https://www.whxunw.com/exam/login.thtml
   - 使用账号密码登录

2. **进入答案页面**
   - 导航到"网络助学-平时作业-作业记录-详情"
   - 或者直接访问答案详情URL

3. **使用插件提取答案**
   - 点击浏览器工具栏中的插件图标
   - 点击"一键提取答案"按钮
   - 查看提取结果
   - 导出为Excel或CSV格式

## 🔧 技术实现

### 核心技术栈
- **Manifest V3** - Chrome扩展最新规范
- **JavaScript ES6+** - 现代JavaScript语法
- **Chrome扩展API** - 使用Chrome原生API
- **CSS3** - 响应式设计，支持主题切换

### 数据提取策略
插件采用多策略数据提取：

1. **DOM遍历策略** - 通过特定class名称查找题目
2. **内容分析策略** - 通过文本特征识别题目和答案
3. **表格解析策略** - 解析HTML表格结构
4. **混合策略** - 结合多种方法提高准确率

### 数据存储
- **chrome.storage.local** - 本地数据存储
- **结构化数据格式** - JSON格式存储题库和历史记录
- **数据加密** - 敏感信息加密存储

## 🐛 常见问题

### Q: 插件为什么无法提取答案？
**A:** 可能的原因：
1. 页面结构发生变化
2. 未登录或登录状态失效
3. 页面加载不完整
4. 网络连接问题

**解决方案：**
1. 刷新页面后重试
2. 重新登录平台
3. 检查网络连接
4. 联系技术支持

### Q: 导出的文件打不开？
**A:** 确保：
1. 使用正确的软件打开（Excel打开.xlsx，文本编辑器打开.csv）
2. 文件没有损坏
3. 文件编码正确（UTF-8）

### Q: 插件支持其他平台吗？
**A:** 目前仅支持讯网教学云平台。未来可能扩展支持其他平台。
