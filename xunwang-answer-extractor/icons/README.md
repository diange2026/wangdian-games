# 插件图标说明

## 必需图标文件

Chrome浏览器插件需要以下尺寸的图标：

### 1. 主要图标
- **16x16** - 浏览器工具栏图标
  - 文件名: `icon16.png`
  - 用途: 浏览器工具栏显示
  - 要求: 清晰可识别，简洁设计

- **48x48** - 扩展管理页面图标
  - 文件名: `icon48.png`
  - 用途: chrome://extensions/ 页面显示
  - 要求: 与16x16版本设计一致，更精细

- **128x128** - Chrome应用商店图标
  - 文件名: `icon128.png`
  - 用途: Chrome Web Store 展示
  - 要求: 高质量，吸引眼球

### 2. 可选图标
- **32x32** - Windows任务栏图标
- **96x96** - 高分辨率显示
- **192x192** - 超大图标

## 设计建议

### 颜色方案
- 主色: 蓝色系 (#4a6ee0) - 代表专业、信任
- 辅色: 紫色系 (#6a11cb) - 渐变效果
- 强调色: 绿色 (#2ecc71) - 表示正确、成功

### 图标元素
建议包含以下元素：
1. **书本** - 代表学习、知识
2. **放大镜** - 代表搜索、提取
3. **对勾** - 代表正确、答案

### 设计规范
1. **简洁性**: 避免过多细节，确保在小尺寸下清晰
2. **一致性**: 所有尺寸保持相同设计风格
3. **可识别性**: 一眼能看出是教育相关工具
4. **适应性**: 适应不同背景色（深色/浅色模式）

## 临时占位图标

由于设计过程尚未完成，目前使用以下占位方案：

1. **生成简单图标**
   - 使用在线图标生成器
   - 使用开源图标库（如Font Awesome）
   - 使用AI工具生成

2. **图标来源推荐**
   - [FlatIcon](https://www.flaticon.com/)
   - [Icons8](https://icons8.com/)
   - [FontAwesome](https://fontawesome.com/)
   - [Material Icons](https://material.io/resources/icons/)

## 图标更新流程

1. **设计图标**
   - 使用设计工具（Figma、Sketch、Illustrator）
   - 确保所有必需尺寸
   - 导出PNG格式

2. **替换图标**
   - 备份原图标
   - 替换新图标
   - 文件名保持不变

3. **测试验证**
   - 在浏览器工具栏测试
   - 在扩展管理页面测试
   - 多分辨率测试

## 注意事项

1. **文件格式**: 必须为PNG格式
2. **透明度**: 建议使用透明背景
3. **文件大小**: 单个图标不超过100KB
4. **命名规范**: 严格按要求命名
5. **位置**: 必须放在`icons/`目录下
